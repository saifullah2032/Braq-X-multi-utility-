## 🚀 Overview

BARQ X is a sophisticated multi-utility gesture control application that transforms your Android device into a powerful gesture-driven command center. Featuring a unique **Mythic Gutter Dashboard** with **10% notebook margin optimization** and **neo-brutalist design philosophy**.

### ✨ Key Features

- **🔥 Mythic Protocol Legend**: Five gesture protocols with mythological naming
- **📱 Neo-Brutalist UI**: Sharp edges, high contrast, 3.5px borders throughout
- **🎯 Optimized Layout**: 10% gutter margin for maximum screen utilization
- **⚡ Real-Time Sensors**: Advanced sensor fusion and gesture recognition
- **🎨 Dynamic Theming**: High-contrast wide-rule notebook background
- **📊 Live Dashboard**: Real-time system status and gesture monitoring

## 🎭 Mythic Protocol System

| Protocol | Icon | Gesture | Action | Status |

## 🛠️ Installation & Setup

### Prerequisites
- **Flutter SDK**: 3.38.5 or higher
- **Android SDK**: API level 21+ (Android 5.0+)
- **Dart**: 3.10.4 or higher

### Quick Start
```bash
# Clone the repository
git clone https://github.com/saifullah2032/Braq-X-multi-utility-.git
cd barq_x

# Install dependencies
flutter pub get

# Run the app
flutter run

# Build release APK
flutter build apk --release --split-per-abi
```

### Required Permissions
- **Sensors**: Accelerometer, Gyroscope access for gesture detection
- **Camera**: For HERMES SNAP protocol
- **Audio**: For system sound control
- **System Settings**: For DND and system toggles
- **Background Processing**: For continuous gesture monitoring
